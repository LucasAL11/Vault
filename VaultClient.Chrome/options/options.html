► Configuracoes Avancadas (HMAC)
ℹ O que e o HMAC? Para revelar o valor de um segredo, o servidor exige uma prova criptografica (HMAC-SHA256). O Client ID e o Client Secret aqui sao credenciais de cliente registadas no servidor — nao sao a chave de assinatura JWT.
⚠ Seguranca Usa credenciais exclusivas para esta extensao. Nunca uses a mesma chave do servidor ou de producao. O valor fica guardado localmente no browser.
